# Offensive Security Toolkit

Personal toolkit and notes developed through hands-on labs (HackTheBox, TryHackMe, etc.)

## Structure

- recon/        → Enumeration techniques and commands
- web/          → Web exploitation notes & payloads
- privesc/      → Privilege escalation techniques
- shells/       → Reverse shells & upgrades
- windows/      → Windows-specific techniques
- linux/        → Linux-specific techniques
- payloads/     → Custom payloads
- tools/        → Custom scripts
- notes/        → General notes & findings

## Philosophy

This repository is built for:
- Repeatability
- Understanding (not just copy/paste)
- Real-world application
